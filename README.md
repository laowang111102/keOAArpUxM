# 漫画推荐系统 python1179

## 项目概述

本项目是一个基于Python开发的漫画推荐系统，主要包含信息分析与爬虫模块，为用户推荐个性化的漫画内容。

## 技术栈

### 后端技术
- Python
- Django框架

### 数据库
- MySQL 5.7.33

### 前端技术
- jQuery
- Bootstrap
- HTML
- JSP

### 开发工具
- PyCharm

## 功能模块

- 用户管理：用户注册、登录、信息管理
- 漫画信息爬取：自动爬取网络上的漫画信息
- 信息分析：对用户行为与漫画内容进行分析
- 推荐系统：为用户提供个性化的漫画推荐

## 系统角色

- 普通用户：浏览、搜索、查看漫画，获取个性化推荐
- 管理员：系统管理、用户管理、漫画信息管理

## 运行环境

- Python环境（建议使用虚拟环境）
- MySQL数据库
- 支持HTML、CSS、JavaScript的浏览器

## 部署步骤

1. 安装Python环境与依赖库
2. 安装MySQL数据库，创建数据库与表结构
3. 配置项目中的数据库连接信息
4. 运行后端Django项目
5. 部署前端静态文件到Web服务器
